About AssayTrace

Change-Control Intelligence for Clinical Genomics

Automated impact analysis, revalidation decisions, benchmark evidence integration, and audit-ready reporting for regulated NGS workflows.

The Problem

Clinical NGS pipelines continuously evolve. Reference genomes change. Annotation databases change. Variant callers change. QC policies evolve.

Each update can affect validated performance claims, interpretive workflows, and the evidence laboratories must defend during inspections. Determining the appropriate scope of revalidation is often manual, inconsistent, time-consuming, and difficult to document in an audit-ready manner.

The Solution

AssayTrace brings structure and traceability to NGS change control. The platform:

  • Detects pipeline changes between baseline and candidate assay manifests
  • Performs impact analysis across analytical, interpretive, quality, and infrastructure domains
  • Maps affected assay claims to each detected change
  • Recommends revalidation actions with explicit, auditable rationale
  • Generates audit-ready documentation, including benchmark regression and variant delta evidence

Core Capabilities

Change Detection

Structured comparison of versioned pipeline components, QC thresholds, reference resources, and environment metadata between two assay manifests.

Impact Analysis

Classifies each change into impact domains — analytical, interpretive, quality, infrastructure, or documentation — to guide validation scope.

Assay Claim Mapping

Links detected changes to the validated assay claims they may affect, using explicit claim dependency declarations in the manifest.

Benchmark Regression Detection

Compares baseline and current benchmark evidence to flag performance regressions that may require analytical follow-up.

Variant Delta Tracking

Surfaces additions, removals, and classification changes in reportable variant sets between pipeline versions.

Audit Binder Generation

Assembles change summaries, impact rationale, revalidation decisions, regression metrics, and variant deltas into a single audit-ready record with PDF export.
